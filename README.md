# Siber Güvenlik Notları

Bu depo, siber güvenlik, tersine mühendislik, ağ analizi ve sistem güvenliği alanlarındaki öğrenme sürecimi, araştırmalarımı ve kişisel notlarımı barındırmaktadır. 

## Hakkında

Yazılım mühendisliği ve düşük seviye programlama (Assembly vb.) temelleri üzerine inşa ettiğim siber güvenlik serüvenimde, edindiğim teorik bilgileri ve pratik uygulamaları burada dokümante ediyorum.
Amacım, karmaşık güvenlik konseptlerini anlaşılır bir şekilde kaydetmek ve ihtiyaç duyduğumda hızlıca başvurabileceğim bir bilgi bankası oluşturmaktır.

## İçerik Yapısı

Notlar şu anda aşağıdaki ana kategoriler altında toplanmaktadır:

* **Ağ Güvenliği:** Protokoller, paket analizi ve ağ tabanlı saldırı/savunma senaryoları.
* **Web Güvenliği:** Yaygın web zafiyetleri üzerine pratik notlar.

## Kullanım

Bu depo kişisel bir referans kaynağı olarak tasarlanmıştır. Notların büyük bir kısmı Markdown (`.md`) formatında hazırlanmış olup, hem GitHub üzerinden hem de Obsidian gibi yerel Markdown editörleriyle kolayca okunabilir.
