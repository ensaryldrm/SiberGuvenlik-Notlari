Dize demektir.

Bir **char** tek bir karakterdir (örneğin: 1, 6, %, b, p, ., T vb.)

**str** (dize) türü, birden fazla **char** içeren özel bir türdür.

Bir değişkende dize değeri başlatmak için onu tek veya çift tırnak işaretleri arasına alın:

```python
s1 = 'This is a string's2 = "This is also a string"
```

## f-string Yapısı

```python
isim = "Ensar"
renk = "Mavi"
print(f"Hoş geldin {isim}, en sevdiğin renk {renk}.")
```
