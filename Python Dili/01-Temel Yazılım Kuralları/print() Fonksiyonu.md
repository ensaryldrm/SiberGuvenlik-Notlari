### 1. Metin (String) Yazdırma
Metinler mutlaka tırnak içinde yazılmalıdır. 
Tek tırnak (' ') veya çift tırnak (" ") kullanabiliriz, Python için ikisi de aynıdır.
```python
print("Merhaba Dünya")
print('Python yazmaya basladim')
```

### 2. Sayı (Integer/Float) Yazdırma
Sayılarda tırnak işareti kullanılmaz. Doğrudan yazılır.
```python
print(100)
print(3.14)
```
### 3. Birden Fazla Değeri Yan Yana Yazdırma
Virgül koyarak farklı türdeki verileri aynı anda yazdırabiliriz.
Python aralarına otomatik olarak bir boşluk koyacaktır.
```python
print("Benim yaşım:", 25)
print("Sistem durumu:", "Aktif", 200)
```
